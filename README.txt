This project is an iOS application written in Swift to perform basic statistics calculations. 
On this first screen, enter your number values separated by a comma with no spaces and click submit. 
On the second screen, the following calculations will display based on your entered values:

Mean
Median 
Mode
Count
Sum
Maximum
Minimum
Range
Geometric Mean
Standard Deviation
Variance
Sample Standard Deviation
Sample Variance

From the second screen, you can then choose to go back to the first page to edit your values or enter new values.
